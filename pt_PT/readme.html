Instalação: Os famosos 5 minutos

  1. Descompacte o pacote num directório vazio e carregue tudo.
  2. Abra o ficheiro wp-admin/install.php no seu navegador. Será guiado no procedimento de criação do ficheiro wp-config.php com os detalhes de conexão à sua base de dados.
    1. Se por alguma razão isto não funcionar, não se preocupe. Não funciona sempre em todos os servidores. Abra o ficheiro wp-config-sample.php num editor de texto como o WordPad ou semelhante e introduza manualmente os dados de acesso à sua base de dados.
    2. Guarde o ficheiro como wp-config.php e carregue-o para o servidor.
    3. Abra agora o ficheiro wp-admin/install.php no seu navegador.
  3. Com este ficheiro criado, o procedimento de instalação irá preparar as tabelas necessárias ao seu site. Se ocorrerem erros, verifique o ficheiro wp-config.php e tente novamente. Se mesmo assim não funcionar, por favor visite o fórum de suporte com toda a informação que conseguir fornecer.
  4. Se não especificar uma senha, tome nota da senha que foi gerada para si. Se não especificar um nome de utilizador, este será admin.
  5. No final, o instalador deverá reencaminhá-lo para a página de início de sessão. Inicie a sessão com o nome de utilizador e senha que escolheu (ou com os que lhe foram atribuidos) durante a instalação. Se foi gerada uma senha poderá clicar em 'Perfil' para mudá-la.

Actualização

Actualizações automáticas

  1. Abra o ficheiro wp-admin/update-core.php no seu navegador e siga as instruções.
  2. Queria mais talvez? Não há!

Actualizações manuais

  1. Antes de actualizar o que quer que seja, verifique que fez cópias de segurança dos ficheiros que modificou, por exemplo index.php.
  2. Elimine os ficheiros antigos do WordPress, guardando os que modificou.
  3. Carregue os novos ficheiros.
  4. Abra /wp-admin/upgrade.php no seu navegador.

Migração de outros sistemas

O WordPress pode importar conteúdo a partir de vários sistemas. Garanta, antes de mais, que a sua instalação está a funcionar antes de usar as nossas ferramentas de importação.

Requisitos de sistema

  • PHP versão 7.2.24 ou superior.
  • MySQL versão 5.5.5 ou superior.

Recomendações

  • PHP versão 8.3 ou superior.
  • MySQL versão 8.0 ou superior, OU MariaDB versão 10.6 ou superior.
  • O módulo Apache mod_rewrite.
  • Suporte para HTTPS.
  • Uma ligação para pt.wordpress.org no seu site.

Recursos Online

Se tiver algumas perguntas sem resposta neste documento, utilize um dos vários recursos online do WordPress:

Comunidade Portuguesa de WordPress
Página da Comunidade Portuguesa de WordPress, com notícias e ajuda em português.
HelpHub
O HelpHub é a enciclopédia de tudo relativo ao WordPress. Trata-se do mais abrangente recurso de informações sobre o WordPress.
O Blog do WordPress
Aqui poderá encontrar as últimas actualizações e notícias relacionadas com o WordPress. As notícias mais recentes aparecem no seu painel, por omissão.
Planeta WordPress
O Planeta WordPress é um agregador de notícias que junta artigos de blogs do WordPress de todo o mundo.
Fórum de suporte do WordPress
Se procurou em todo o lado e mesmo assim não consegue encontrar um resposta, experimente procurar ajuda no fórum de suporte que é suportado por uma ampla comunidade, muito activa e pronta a ajudar. Para que os outros possam ajudar de forma eficaz, certifique-se de que utiliza um título descritivo e que apresenta o seu problema com o maior detalhe possível.
Canal de IRC do WordPress
Por último, existe um canal de conversação online que é utilizado para debate entre pessoas que utilizam o WordPress e, ocasionalmente, para tópicos de suporte. A página wiki acima deverá fornecer-lhe as orientações necessárias. (irc.libera.chat #wordpress)

Notas finais

  • Se tiver alguma sugestão, ideia, comentário ou se encontrar um bug (oh não!), partilhe essa informação no Fórum de suporte.
  • O WordPress tem uma API robusta, que torna a expansão do código muito fácil. Se for um programador interessado em utilizar esta funcionalidade, consulte a documentação do Plugin Developer Handbook. Na esmagadora maioria dos casos, não deverá modificar o código principal.
